# Atividades HTML, CSS & JavaScript

Repositório com exercícios práticos de HTML, CSS e JavaScript, organizados por aula. Cada atividade é uma página independente, desenvolvida com JavaScript puro (sem frameworks ou bibliotecas).

## Sobre

Projeto pessoal desenvolvido para fins de estudo e portfólio, como parte do meu aprendizado em desenvolvimento Fullstack.

© 2026 Gimenes. Todos os direitos reservados.

## Conteúdo

O repositório reúne atividades de três frentes:

**HTML**
- Estruturação semântica de páginas
- Formulários e elementos de input

**CSS**
- Estilização e layout
- Responsividade

**JavaScript**

*Fundamentos*
- `alert`, `prompt`
- Estruturas de repetição: `while`, `do while`
- Estrutura condicional `switch`
- Funções básicas

*Intermediário*
- Manipulação do DOM
- Eventos (`click`, `input`, `blur`, etc.)
- Validação de formulários
- Expressões regulares (regex)
- Máscaras de input (CPF, telefone, etc.)
- Lógica de programação aplicada a problemas reais

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Como executar

Não é necessário nenhum servidor ou instalação. Basta abrir o arquivo `.html` desejado diretamente no navegador.

## Estrutura do repositório

Os arquivos são organizados por aula, em pastas separadas. A pasta `fundamentos/` reúne as atividades mais básicas (alert, prompt, laços de repetição, switch e funções simples):

```
Atividades-HTML-CSS-JavaScript/
├── fundamentos/
│   ├── 1-exemplo.html
│   └── ...
├── aula-13/
│   ├── 1-vogal.html
│   ├── 2-media-notas.html
│   └── ...
├── aula-14/
│   ├── ...
├── aula-15/
│   ├── ...
└── README.md
```

Cada pasta de aula reúne os exercícios propostos naquela data, mantendo o histórico de evolução do aprendizado.
